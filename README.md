# Zettelkasten with Obsidian - simplified version for researchers

- This has been made to help my friends. Sorry that it's been roughly made, but you can make your own referencing this as an example.
- This is based on Gropl's work. I appreciate his work. You can reference his page for further information. https://github.com/groepl/Take-Useful-Notes 

# Plugins
- Go to setting > Community plugins > Community plugins-Browse
1) Canvas Mindmap
2) Dataview
3) Excalidraw
4) Frontmatter Tag Suggest
5) TimeStamper
6) Update time on edit

# How to use
### 1_Fleeting Notes
- You can put random notes, such as notes from a seminar or ideas

### 2_Literature Notes
- Notes with summaries from various sources
- Sub-folders are not recommended
- 'Inbox' folder is for notes that are being worked on

### 3_Permanent Notes
- Each note has only one concept
- Sub-folders are not recommended
- 'Inbox' folder is for notes that are being worked on

### 4_Category Notes
- Categorize permanent notes to find notes by categories

### 5_Project Notes
- This is for projects such as a thesis

### Gardening
- This can help you clean your files

### Templates
- Use templates for consistent formates
